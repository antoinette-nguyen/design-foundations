# Design Foundations

A 9-month public learning project documenting my growth from UI/UX designer to confident frontend developer.

## Mission

I'm a designer who decided to close the gap between what I can imagine and what I can build. This repo is where I develop the foundational HTML, CSS, and Git skills that every great frontend designer needs. Every commit is a step toward a portfolio that reflects both how things look *and* how they're made.

## What's in here

- Semantic HTML layouts built from scratch
- CSS components without any frameworks
- A design token system (colors, spacing, typography)
- A static portfolio landing page — the first version of many

## Timeline

**Phase 1 — Foundations** | March–May 2026
Building comfort with HTML, CSS, Git, and introductory React. Every lesson produces a real file in a real repo.
